# Day 1 - Eligibility Checker

## Project Description

A simple Python console application that checks a person's eligibility for:

- Voting
- Driving Licence
- Marriage

based on age and gender.

## Concepts Used

- Variables
- Input / Output
- Data Types
- Type Casting
- Comparison Operators
- Logical Operators
- Nested if
- if-elif-else
- f-strings
- String Formatting
